# Ruptiva Backend Test

Teste realizado utilizando node, typescript, typeorm, postgres 

## Installation

Yarn or Npm

```bash
yarn

yarn typeorm migration:run

yarn typeorm migration:generate -- -n {"Nome que escolher"}
```

## Start

```python
yarn dev:server
```
